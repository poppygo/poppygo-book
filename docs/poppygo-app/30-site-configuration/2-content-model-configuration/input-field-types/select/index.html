      Select #

      The select field generates a dropdown selectbox for selecting strings or numbers. The output is a number or string. If multiple is set true the out is an array of numbers or strings.

      Properties #

      propertyvalue typeoptionaldescription
      keystringmandatoryKeys are for internal use and must be unique
      titlestringmandatoryThe title of the element
      multiplebooleanoptional (default false)Enable multiple selection
      optionsarray of dictionariesmandatoryArray with title/value pairs
      options.[n].textstringmandatoryString with option visible text
      options.[n].valuestringmandatoryString with option value to save when selected

      Sample #

      Configuration #

        key: sample_field
  title: Sample field
  type: select
  multiple: false
  options:
    - text: Option 1
      value: 1
    - text: Option 2
      value: 2
    - text: Option 3
      value: 3

      Output #

      sample_field: 2
